Remove Management Server Failover

Target: Support (Makes changes to the system)

Usage: This program removes the Milestone XProtect Management Server Failover feature from the system. This is used when the failover feature is no longer needed, or if the failover feature is causing problems with the system.

Usage: Run on the Management Server of each node. Simply click the Remove button. When complete, click the Open Log buttone to view the log file. The log file will contain information about the changes that were made to the system, as well as any errors that were encountered during the process.

Should be Run as Administrator.
